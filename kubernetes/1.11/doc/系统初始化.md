1.安装Docker

第一步：使用国内Docker源

    cd /etc/yum.repos.d/
    wget https://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
第二步：Docker安装：
    yum install -y docker-ce
第三步：启动后台进程：

     systemctl start docker
2.准备部署目录

    mkdir -p /opt/kubernetes/{cfg,bin,ssl,log}
3.设置hosts解析

    vim /etc/hosts
